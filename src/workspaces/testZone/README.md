# TestZone Workspace Module

## Описание

Модуль TestZone предоставляет простые тестовые workspace для разработки и тестирования layout системы. Используется для проверки работы split containers и динамического создания workspace.

## Структура модуля

```
workspaces/testZone/
├── TestZoneView.tsx    # Компонент тестовой зоны
├── index.ts            # Регистрация всех test типов
└── README.md          # Документация модуля
```

## Зависимости

### Core
- `@core/types/workspace` - WorkspaceId

### Shared
- Нет зависимостей от shared модулей

## Функциональность

### Основные возможности

1. **Визуальное отображение**
   - Отображение ID workspace
   - Отображение ID зоны
   - Простой UI для идентификации зоны

2. **Тестирование layout**
   - Проверка работы split containers
   - Проверка динамического создания workspace
   - Проверка регистрации модулей

### Особенности реализации

- Минимальная реализация для тестирования
- Использует CSS переменные для стилизации
- Не имеет функциональности, только визуальное представление

## Регистрация

Модуль автоматически регистрируется при импорте в `src/index.tsx`:

Все 8 типов (test1-test8) регистрируются автоматически.

## Использование

Модуль используется автоматически через `WorkspaceRenderer` для workspace с типами `'test1'` - `'test8'`.

Обычно используется в development режиме для тестирования сложных layout конфигураций.

## Примечания

Этот модуль предназначен только для разработки и тестирования. В production версии может быть удален или заменен на реальные модули.
